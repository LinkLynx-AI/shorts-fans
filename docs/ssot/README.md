# short-fans SSOT

This directory is the shared entry point for the `short-fans` planning docs.

## What this folder contains

- `index.md`
  - top-level SSOT map
- `product/`
  - product structure, UX, account model, moderation, creator workflow, MVP scope
- `business/`
  - market, creator acquisition, revenue, audio, data strategy
- `refs/`
  - reference links and supporting source pointers

## Recommended reading order

1. `index.md`
2. `product/index.md`
3. `business/index.md`
4. `product/scope/mvp-boundaries.md`
5. `product/fan/core-experience.md`

## Current core thesis

- `short` is the public surface
- `main` is the paid continuation that starts from the same context
- the core loop is `feed -> short -> unlock -> main`
- initial distribution assumes `PWA / web`
- initial market assumes the largest English-speaking market first
- the first monetization candidate is `pay-per-unlock centric`

# short-fans Business SSOT - Market And Strategy

## 位置づけ

- 市場の切り方、競合観、差別化、スケール仮説をまとめる
- 現時点では確定事項よりも、`どこを深掘るか` の軸を明確にすることを優先する
- creator 獲得チャネルや運用詳細は `creator-acquisition-strategy.md` を canonical にする

## 安定した事実

- 事業面で優先的に調べる論点として、`市場分析`、`国別分析`、`収益構造`、`プロダクト別ユーザー数`、`カテゴリ別構成比` が上がっている
- 想定 creator は、`Instagram` や `YouTube Shorts` などへ普段から short を出している `個人〜少人数チーム`
- 競合観察の基本セットは `OnlyFans / Fansly / Fanvue / myfans / Fantia`
- `Fansly` は `FYP + free preview + locked conversion` を公式に持っている
- `Fanvue` は `Search & Discover`、`merchant of record`、`AI-friendly policy` を持つ
- `myfans` は `サンプル動画必須`、`creator dashboard`、`premium creator優遇` を持つ
- `Fantia` は `公開コンテンツ + 限定コンテンツ` の構造を持ち、実写カテゴリは `17.5%` fee で運用している
- `Semrush` の 2026-01 / 2026-02 推計では、`OnlyFans / Fansly / Fanvue` は米国流量が最大、`myfans / Fantia` は日本流量が最大
- `public preview + recommendation + paid conversion` 自体は、海外では既に競争領域である
- 初期 market は `英語圏最大市場` を優先する
- 初期 creator target は `英語圏の OnlyFans系 short 強者` とする
- `配信者 / コスプレ / 日本ローカル creator` は初期の主 target には置かない
- `成人向けど真ん中` で行くが、`short 面` は露骨なポルノにしない

## 継続仮説

- 差別化の中心は、`公開 short feed があること` 単体ではなく、`short -> canonical main -> unlock` を一貫設計すること
- `TikTok / Reels` 的な public surface は依然重要だが、それだけでは `Fansly` との差が弱い
- creator supply は、既に short を作れる creator を取り込む方が立ち上がりやすい
- 事業の初期スケールでは、`コンテンツ供給の質と量` が需要より先にボトルネックになる可能性が高い
- 国別優先順位は、単純な需要だけでなく `決済通りやすさ` と `審査 / 規制` を合わせて判断する必要がある
- `AIシーズニング` は `プロダクト機能` ではなく、`初期の content supply / growth strategy` として見るべき可能性が高い
- `英語圏先行` は競合が最も激しい一方、最大市場を狙うという方針と整合する
- 約束価値は `かっこいい short 中心の成人向け面` と `main unlock の売上面` を両立させることに置く

## 未解決論点

- `英語圏` の中で `US中心`、`北米英語圏`、`より広い英語圏` のどこから入るか
- `TAM / SAM / SOM` をどう切るか
- LP で何を約束価値として前面に出すか
- `OnlyFans / Fansly / Fanvue / myfans / Fantia` のどこを主ベンチマークに置くか
- AI コンテンツと real creator コンテンツの比率をどう管理するか
- `かっこいい` brand / visual と `adult-first` をどう両立させるか
